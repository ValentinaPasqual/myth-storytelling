Telling Aeneis through museal art

In mythLOD collection, each museal artefact has been analysed by an expert, which defined what the artefact represents (conceptual category). Additionally, to some of these interpretations the expert added some literary references in which the scene is depicted. By analysing these aspects, we found out that Aeneis by Virgil is one of the most referred literary source in mythLOD dataset.

In this page the Aeneis are presented from mythLOD perspective. Additionally, the available visualisations have been grouped into What (representing artefacts, literary sources and categories), Where (e.g. places where the items are located), When (e.g. date of an artefact creation), Who (representing people involved into the creation of artefacts, literary sources and interpretations).

what

With "WHAT" we refer to all the cultural objects represented in mythLOD assertions, namely:

  • Museal Objects, which are actually the center of mythLOD collection;
  • Conceptual Categories, which are the result of the interpretation of each cultural object by an expert (e.g. what this painting represents?)
  • Literary References, which are all the references which share a category with each mythLOD museal object (e.g. Aeneis)

In this section, these three aspects are presented, considering them both separately and interconnected between them.

Moreover, Aeneis can be considered both a set of cited lines (as we have seen in the heamap visualisation) but also as a whole literary source (frbr:F1_Work).

In each mythLOD assertion, a museal object shares one or more common themes (conceptual categories) with one or more literary sources from different authors and periods. And so, which are all the literary references which co-occur with Aeneis?

The result is the network here presented, which shows the literary sources and categories connection. What emerges, is the pink set showing that Aeneis shows most categories mainly with four other works:

  • Petrarca Francesco, 1304-1374 | Trionfi
  • Ungaretti Giuseppe | Vita di un Uomo
  • Dante Alighieri, 1265-1321 | Divina Commedia
  • Leopardi Giacomo, 1798-1837 | Divina Commedia
